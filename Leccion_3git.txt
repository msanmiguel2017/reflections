14. Forks y clone son dos operaciones iguales, la diferencia es que forks se hace en el servidor de Github para adquirir un repositorio y clone se hace del servidor de github al portatil
Branch se hace en el portatil y sirve para hacer ediciones personales en el repositorio las cuales pueden adicionarse luego al master.
Fork la uso en Github
Clone la uso entre Github y mi portatil
Branch la uso sobre el repositorio en mi portatil
22. Es muy importante llevar un orden en los nuevos commits para que sea mas facil hacer nuevos cambios en el repositorio y evitar cualquier perdida de informacion. Tambien sirve para que las adiciones de los colaboradores se incluyan de manera favorable al repositorio.
25. Como colaborar sin usar Git o Github? Encontrar un sistema similar de compartir archivos o mandar archivos de las cosas que cambian.
